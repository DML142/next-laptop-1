## What is this && how to run

Non commercial open source pet-project build to demonstrate my potential in frontend development, also it's design from official Apple Macbook-Pro site rewritten by me from scratch, and so yeah. Install with ```git clone``` and ```npm i```, run with ```npm run dev``` and thats it.

#Which tools I used?

Npm, react, lucide-react, nextjs, gsap, tailwindcss, postcss, eslint and typescript
